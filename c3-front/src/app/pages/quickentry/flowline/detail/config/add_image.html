添加镜像方法
1.修改本地配置，添加镜像仓库地址：
 在/etc/docker/daemon.json添加下面内容 { "registry-mirrors": ["https://registry.docker-cn.com"], "insecure-registries": ["10.10.10.10:5000"] }

2.添加用户名密码：
 到 http://job.open-c3.org/#/work/common/14496 执行”添加docker仓库用户“作业添加用户名密码

3.docker登录：
 docker login -u 用户名 -p 密码 10.10.10.10:5000

4.打tag：
 如： docker tag docker.io/centos:6 10.10.10.10:5000/centos:6

5.上传镜像：
 如： docker push 10.10.10.10:5000/centos:6

6. 发布镜像到构建系统：
 到 http://job.open-c3.org/#/work/common/14496 执行“从镜像仓库把镜像同步到国内CI”作业，image名就是要发布的名字

7.登出：
 docker logout
